🤖 AI Text Transformer

An AI-powered text transformation web application built using React, Vite, and Google Gemini API. This tool helps users quickly improve, rewrite, and transform text into different formats.

🚀 Features

- 📝 Text Summarization
- ✅ Grammar & Spelling Correction
- 💼 Professional Tone Conversion
- 😊 Casual Tone Conversion
- 📋 Copy Output to Clipboard
- 📊 Word & Character Counter
- 📱 Responsive Design
- ⚡ Fast AI-Powered Responses

🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Vite
- CSS3
- Google Gemini API
- Git & GitHub
- Vercel

📸 Preview

Enter any text and transform it instantly using AI-powered features such as summarization, grammar correction, professional tone, and casual tone conversion.

⚙️ Installation

git clone https://github.com/rajeshxdev/AI-Text-Transformer.git
cd AI-Text-Transformer
npm install
npm run dev

🔑 Environment Variables

Create a ".env" file in the project root:

VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
